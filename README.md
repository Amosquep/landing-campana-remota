# Landing Page: Campaña Informativa

Este proyecto es una landing page diseñada como parte de la Actividad de Aprendizaje #1 del programa ADSO en el SENA.

## Estructura del Proyecto
El proyecto sigue una organización estándar para facilitar el mantenimiento y escalabilidad:

- `index.html`: Página principal de la campaña.
- `css/`: Contiene los estilos visuales del sitio (`estilos.css`).
- `js/`: Contiene los archivos de lógica y dinamismo (`script.js`).
- `img/`: Carpeta destinada a los recursos gráficos del proyecto.
